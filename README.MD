# Fornato de enfasis
- letra *italica* de la primera forma.
- letra _italica_ de la segunda forma.
- formato **bold o strong**.
- formato __bold o strong__ de la 2º forma.
- formato ~~tachado~~.
- Aqui podemos usar *formato italico*, pero también **bold** y ~~tachado~~
